# dongKBO
- KBO(한국 프로야구)의 팀 순위 및 개인 기록과 순위를 보여주는 홈페이지

# 사용 언어
- Java(JDK 1.11), HTML/CSS, JavaScript
- Apache Tomcat 9.0
- Spring Framework 5.0.7
- MySQL
- RESTfulAPI(JSON), Jquery
